## 🛠️ Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/heyraqueell/Java-iniciante.git`
2. Abra a pasta no **IntelliJ IDEA**.
3. Certifique-se de ter o **JDK 17** ou superior instalado.
4. Navegue até a pasta `src` e execute a classe desejada.
